# Craftsman Property Enhancements

Static, Docker-ready website for Craftsman Property Enhancements.

## Run locally with Docker

```bash
docker compose up --build
```

Then open:

```text
http://localhost:8080
```

## GitHub Pages

This is a static HTML/CSS/JS website. Upload the full folder contents to a GitHub repository and enable GitHub Pages from the repository settings.

## Structure

- `index.html` — homepage
- `services.html` — property refresh and repairs
- `outdoor.html` — outdoor grounds and landscaping improvements
- `compliance.html` — pool and hot tub compliance
- `about.html` — company background
- `contact.html` — contact page
- `css/style.css` — full light-theme styling
- `js/main.js` — mobile navigation and footer year
- `images/` — uploaded brand and website images
- `Dockerfile` and `docker-compose.yml` — Docker deployment
